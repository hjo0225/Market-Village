**M4 — 미니 리더보드(§13.7)는 무상태 파생으로 지금 짓는다** 🤖 council (플레이어경험 1위 체감 + 리스크 렌즈 안전 설계)

**M5 — 결정론 안전 규칙(D11 확장)** 🤖 council (리스크 렌즈 R1~R10 채택)
